### 2.5 Skill 索引 + 角色绑定 (对应总纲 5.1 / 5.2)

### 2.6 策略引擎 (对应总纲第三章)

- L2 策略拦截：Plan 模式拒绝危险工具，Ask 模式弹窗确认，Bypass 放行
- 前端 `StrategyToggle.vue` + `StrategyIndicator.vue`

### 2.7 调度引擎 (对应总纲第四章)

```typescript
// apps/server/src/agent/outer.ts   (416 行)
// apps/server/src/agent/inner.ts   (350 行)
```
- 外层循环（`sessionLoop`）：身份加载 → 会话状态 → 主循环（组装 Prompt → innerLoop → Sub-agent 信号 → 上下文压缩）
- 内层循环（`innerLoop`）：LLM 推理 → Tool Call 检测 → 两阶段校验 → 执行 → 结果追加
- Doom Loop 检测（`detectDoomLoop`）：连续 6 次失败或重复模式时发送系统警报
- 上下文压缩检测

### 2.8 Sub-agent 机制 (对应总纲第四章)

```typescript
// apps/server/src/agent/sub-agent.ts   (181 行)
```
- `delegate_task` 工具定义（task / target_character_id / sub_strategy / instances）
- `validateSubAgentTarget()`：组校验（同组）+ 角色校验（sub/both）
- `spawnAndRunSubAgent()`：创建子会话 → 递归深度限制（MAX_DEPTH=3）→ depth≥1 移除 delegate_task → 调用 innerLoop → 结果回收
- `summarizeAndMerge()`：多实例结果压缩合并

### 2.9 MCP 工具执行 (对应总纲 5.3 / Phase 4.5)
