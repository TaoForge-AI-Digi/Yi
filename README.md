# 弈 (Yì) — AI Agent 系统

> 弈者，坐照全局，执棋而不执子；授势于AI，行其意而不尽其言。庙算于先，机变于后，方寸之间，自有山河。

**弈**是一个融合了人格、策略、工具与知识库的 AI Agent 系统。它的理念来自围棋——人类布大局，AI 落子执行，双方在对弈中互相成就。

## 核心特征

- **有灵魂**——每个 Agent 有独立人格（Soul）、记忆，用户可与不同角色对话
- **有策略**——Plan（只读分析）/ Ask（执行前确认）/ Bypass（自主执行）三种模式手动切换
- **有分身**——主 Agent 可委托 Sub-agent 并行处理子任务，结果自动回收
- **有工具**——Agent 携带 read/write/grep/bash/webfetch 等工具，可绑定白名单约束
- **会进化**——成功执行轨迹可沉淀为 Skill，经验不流失

## 架构

```
apps/
├── client/          Vue 3 前端 — Chat 界面、角色/模型选择、设置面板
└── server/          Hono + Socket.IO 后端 — 双循环 Agent 引擎、LLM 通信、工具执行
```

**服务端核心引擎**采用双层循环架构：外层（Session Loop）管理会话状态、上下文压缩与 Sub-agent 信号；内层（Processor Loop）处理 LLM 推理、工具调用与策略拦截。

## 快速开始

```bash
# 第一步：安装依赖并构建前端
setup.bat

# 第二步：启动服务端和前端
run.bat
```

`setup.bat` 依次安装 server、client 依赖并构建前端；`run.bat` 同时启动服务端（:3001）和前端开发服务器（:5173）。

## 手动启动

```bash
# 终端 1 — 启动服务端
cd apps/server
npx tsx src/index.ts

# 终端 2 — 启动前端开发服务器
cd apps/client
npx vite
```

依赖：Node.js 18+，SQLite（better-sqlite3 内置）。
